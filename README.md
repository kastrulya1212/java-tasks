# Процесс запуска приложений (лабораторных)
Каждая лабораторная имеет свою папку по номеру зантия (lab_1, lab_2, ../). В каждой такой папке лежат классы используемые в этой лабораторной (1 файл = 1 классс), точкой входа является класс называемый "Task_1",где цифра означает номер варианта (во всех лабораторных 1 вариант). Для запуска просто необходимо запустить главные файл
